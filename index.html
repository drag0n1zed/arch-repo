drag's Arch Repo

⚠️ x86_64 ONLY

1a. Add to /etc/pacman.conf

Paste this block at the end of your config:

[drags-repo]
SigLevel = Optional TrustAll
Server = https://repo.zerodrag.dev/$arch

1b. Or One-Liner Install

Run this to append automatically:

sudo bash -c 'echo -e "\n[drags-repo]\nSigLevel = Optional TrustAll\nServer = https://repo.zerodrag.dev/\$arch" >> /etc/pacman.conf'

2. Sync Repositories

Update your database to see packages:

sudo pacman -Sy
